# SKILL: zzz_terminos
> Glosario completo de Zenless Zone Zero: equivalencias Inglés / Chino → Español.
> Usar cuando alguien mencione términos del juego en inglés o chino para responder en español correcto.

---

## REGLA GENERAL DE USO

Cuando un usuario use un término en inglés o chino de ZZZ, **siempre responde con el término en español oficial del juego** (versión ES-ES de HoYoverse). Si no hay término oficial establecido, usar la convención de la comunidad hispanohablante de ZZZ.

## ARGOT DE LA COMUNIDAD (EN → ES)

| Término EN | Español | Explicación |
|---|---|---|
| **DPS** | **DPS** / **Daño** | el personaje que hace el daño principal |
| **Sub-DPS** | **DPS secundario** | hace daño pero no es el principal |
| **Support / Supp** | **Support** / **Auxiliar** | buffa o ayuda al equipo |
| **Healer** | no existe en ZZZ | ZZZ no tiene healers, hay shields y parries |
| **Hyper carry** | **Hyper carry** | equipo construido para un solo DPS |
| **Quickswap** | **Quickswap** | equipo donde cambias personajes constantemente |
| **Anomaly comp** | **Comp de Anomalía** | equipo basado en anomalías |
| **Stun comp** | **Comp de Aturdimiento** | equipo para aturdir al enemigo |
| **On-field** | **En campo** | el personaje que estás controlando activamente |
| **Off-field** | **Fuera de campo** | personaje que actúa desde fuera |
| **Rotation** | **Rotación** | el orden de uso de habilidades del equipo |
| **F2P** | **F2P** / **Gratis** | free-to-play, sin gastar dinero |
| **Whale** | **Ballena** | jugador que gasta mucho dinero |

| **Farming / Grind** | **Farmear** | conseguir recursos repetidamente |
| **Build** | **Build** | la configuración de discos y W-Engine de un personaje |
| **Cope pull** | **Tirar con fe** | tirar por un personaje sin recursos suficientes |
